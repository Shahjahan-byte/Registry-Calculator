## 📖 About The Project

Navigating property transactions can be overwhelming with varying local tax rates and shifting policies. The **Registry Calculator** is a dedicated, web-based utility built to instantly compute the exact costs associated with property registration in Pakistan. 

Whether you are calculating the latest stamp duties for a commercial plot in Lahore or figuring out Capital Value Tax (CVT) and TMA fees across Punjab, this tool takes the guesswork out of real estate planning. It serves as an excellent resource for real estate agents, buyers, and legal consultants who need reliable numbers on the go.

## ✨ Key Features

- **📊 Comprehensive Tax Breakdown:** Instantly calculates Stamp Duty, Capital Value Tax (CVT), Registration Fees, and Town Municipal Administration (TMA) taxes.
- **⚖️ Filer vs. Non-Filer Logic:** Automatically adjusts advance tax rates depending on the buyer/seller's active taxpayer status with the FBR.
- **⚡ Zero Server Lag:** Built purely with frontend web technologies, meaning all calculations happen instantly in the browser.
- **📱 Responsive UI:** Clean, intuitive layout designed to look great on both desktop monitors and mobile screens.

## 💻 Tech Stack

- **HTML5:** Semantic architecture.
- **CSS3:** Custom styling for a professional brand identity.
- **Vanilla JavaScript:** Fast, client-side mathematical computations.
